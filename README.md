# International Football: 150 Years of Data Analysis ⚽📊

## Project Overview
This project explores a dataset of 47,000+ international football matches (1872–2024). I performed an end-to-end EDA and a deep-dive career analysis of Lionel Messi.

## Key Technical Skills
* **Data Cleaning:** Mapping historical to modern successors.
* **Statistical Analysis:** Univariate (Goal Timing) & Bivariate (Performance Evolution).
* **Predictive Insights:** Utilizing Forecast Worksheets for trend projection.
* **Tools:** Python (Pandas, Seaborn, Matplotlib), SQL, Excel.

## Major Findings
* **Home-Field Advantage:** Statistically validated a ~50% win rate for home teams.
* **Transformatory Period:** Identified a 40% performance spike in Messi's peak years (2012-2018).
